# Day 1 - Introduction to MySQL

On Day 1, I learned the basics of databases and the MySQL database management system. 
I understood the difference between DBMS and RDBMS and explored the features of MySQL. 
I installed MySQL Server and MySQL Workbench on my system. 
I also learned how to connect to the MySQL server and execute basic commands. 
This session built a strong foundation for learning SQL.

# Day 2 - Database and Table Creation

On Day 2, I learned how to create, use, and manage databases in MySQL.
I practiced creating tables with different data types and explored database management commands. 
I also learned how to view table structures using the `DESC` command.
Hands-on practice helped me understand how databases and tables are organized. 
This session strengthened my understanding of MySQL fundamentals.

# Day 3 - SQL Fundamentals

On Day 3, I learned the fundamentals of SQL and its different command categories, including DDL, DML, DCL, TCL, and DQL. 
I practiced creating databases and tables using SQL commands. 
I also explored ALTER TABLE operations to modify table structures. 
This session improved my understanding of SQL syntax and database schema management. 
It provided a solid foundation for writing SQL queries.

# Day 4 - ALTER TABLE Operations

On Day 4, I focused on modifying existing database tables using ALTER TABLE commands. 
I practiced adding, modifying, renaming, and deleting columns in tables. 
I also learned about advanced MySQL data types such as ENUM, SET, and DECIMAL. 
These exercises helped me understand schema modifications in real-world databases. 
This session improved my database design skills.

# Day 5 - DML Commands

On Day 5, I learned how to manipulate data using SQL DML commands. 
I practiced inserting, updating, deleting, and retrieving records from database tables. 
I also explored conditional queries using the WHERE clause and learned about COMMIT operations. 
These practical exercises improved my confidence in managing database records. 
This session strengthened my SQL querying skills.

# Day 6 - Constraints and AUTO_INCREMENT

On Day 6, I learned about important MySQL constraints such as PRIMARY KEY and AUTO_INCREMENT. 
I practiced creating tables with automatically generated IDs and explored generated columns. 
I also performed basic CRUD operations on sample tables. 
These exercises helped me understand how constraints improve data integrity. 
This session enhanced my database design knowledge.

# Day 7 - Transaction Control

On Day 7, I learned how transactions work in MySQL to maintain data consistency. 
I practiced using START TRANSACTION, COMMIT, ROLLBACK, and SAVEPOINT commands. 
I also explored table locking and row-level locking using LOCK TABLES and SELECT FOR UPDATE. 
These concepts are important for handling concurrent database operations. 
This session improved my understanding of transaction management.

# Day 8 - SQL Expressions and Calculations

On Day 8, I learned how to perform calculations using SQL expressions. 
I created a Products table and practiced writing queries with arithmetic operations. 
I also calculated taxes, discounts, and used aliases to improve query readability. 
These exercises showed how SQL can perform calculations directly within queries. 
This session strengthened my skills in writing dynamic SQL queries.
